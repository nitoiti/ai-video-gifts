# AI Видео Подарки - Лендинг для создания видео из старых фотографий

Русскоязычный лендинг для сервиса, который создает AI-видео из старых фотографий в качестве подарков для родителей старше 60 лет.

## 🎯 Особенности проекта

- **Полностью на русском языке** - весь контент и интерфейс адаптирован для русскоязычной аудитории
- **Современный дизайн** - использован Tailwind CSS для создания стильного и отзывчивого интерфейса
- **Интерактивная загрузка фото** - drag & drop функциональность с предпросмотром
- **Система тарифов** - три тарифных плана с разными возможностями
- **Модальное окно оплаты** - интуитивный процесс оплаты с выбором способа платежа
- **Адаптивный дизайн** - корректно работает на всех устройствах

## 🚀 Как использовать

1. **Откройте index.html** в любом современном веб-браузере
2. **Просмотрите лендинг** - все разделы доступны для навигации
3. **Попробуйте загрузить фото** в разделе "Попробуйте сейчас"
4. **Выберите тарифный план** и перейдите к оплате (демо-режим)

### Развертывание:
