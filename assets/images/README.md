## 🚀 How to use in HTML:

```html
<!-- Local image -->
<img src="assets/images/hero/demo-video-poster.jpg" alt="Demo Video">

<!-- For GitHub Pages, use relative path -->
<img src="./assets/images/hero/demo-video-poster.jpg" alt="Demo Video">
```

## 📏 Image Guidelines:

- **Format**: JPG, PNG, SVG
- **Size**: Optimized for web (max 500KB)
- **Dimensions**: As needed for each section
- **Alt Text**: Always include descriptive alt text
